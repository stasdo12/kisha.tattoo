ЛОГІКА ПОВЕДІНКИ (для Claude Code)
text
STATE: menu (закрито)
- overlay:    display: none / opacity: 0 / visibility: hidden
- button:     показує [ Menu ]
- body:       scroll — нормальний

STATE: menu-open (відкрито)
- overlay:    display: block / opacity: 1 / visibility: visible
- button:     показує [ Close ]
- body:       overflow: hidden (заблокований скрол)
- overlay:    позиціонується від top = висота хедера (56px total = 20px top + 24px height + 12px padding)

Implement menu / menu-open states for kisha-tattoo project.
Pixel-perfect from Figma.

=== DESIGN TOKENS ===
--color-black: #0D0D0D
--color-white: #FFFFFF
--color-bg-menu-open: #F2F2F2
--font-family: "PP Neue Montreal", sans-serif

=== HEADER (logo-wrapper) ===
DESKTOP:
  position: absolute or sticky
  top: 20px
  left: 40px
  width: calc(100% - 80px)  [1840px on 1920]
  height: 24px
  display: flex
  justify-content: space-between
  align-items: flex-start
  border-bottom: 2px solid #0D0D0D
  padding-bottom: 12px

MOBILE (≤390px):
  top: 20px
  left: 15px
  width: calc(100% - 30px)  [360px on 390]
  same flex rules

Header contains: [Logo SVG left] [Toggle button center] [Tattoo text right]

Toggle button styles:
  font: PP Neue Montreal 500 14px
  line-height: 100%
  letter-spacing: -2%
  color: #0D0D0D
  background: none
  border: none
  cursor: pointer
  States: "[ Menu ]" closed → "[ Close ]" open

=== MENU OVERLAY ===
  position: fixed
  top: 56px  (20 + 24 + 12 = header bottom)
  left: 0
  width: 100vw
  height: calc(100dvh - 56px)
  background: #FFFFFF (desktop) / #F2F2F2 (mobile)
  z-index: 90
  overflow: hidden

Closed state:
  opacity: 0
  visibility: hidden
  pointer-events: none

Open state:
  opacity: 1
  visibility: visible
  pointer-events: auto

=== NAV LINKS ===
DESKTOP:
  position: absolute
  top: 72px
  left: 40px
  display: flex
  flex-direction: column
  gap: 20px

  Link typography:
    font: PP Neue Montreal 500 Medium

    letter-spacing: -2%
    color: #0D0D0D

=== LANGUAGE SWITCHER ===

  Items: En / De / Ua
  Active (En): has border/outline
  Font: same 14px PP Neue Montreal

=== BEHAVIOR ===
On toggle click:
  1. Toggle class "menu-open" on <body> or overlay
  2. Change button label Menu ↔ Close
  3. Toggle overlay visibility
  4. Lock/unlock body scroll (overflow: hidden)

Keyboard:
  Esc key → close menu
  button aria-expanded="true/false"
  button aria-controls="menu-overlay"
  overlay id="menu-overlay" role="dialog"
  Focus trap inside overlay when open
